### 개발 목표

이미지에 마우스를 올렸을 때 (hover), 이미지에 변화가 생기는 사이트 개발

### 사용 기술

- Media Query

### 개선사항

1. 대부분의 div 박스나 img가 %단위로 적혀있어서 높이나 너비에 영향을 많이 받는다. 이 부분을 고려해서 만들어야 한다.
2. 주석을 최대한 많이 달아주면서 개발하자.
3. 중복되는 css 코드는 뼈대와 살을 분리하면서 효율적으로 코드를 짜자
